AjaxAnywhere

is an Open Source declarative Ajax framework for the JVM. It enables the developer to provide async server side rendering and page refresh without Javascript coding.

Declarative approach

It allows for more readable, more maintainable and less error prone code by removing all the boiler plate Javascript, therefore making the developer more happy :-) and productive.

Web Framework agnostic

It integrates seamlessly with any of your favourite Java or JVM Action based Web Frameworks. The learning curve is almost none existent.

Start using it now!!!

AjaxAnywhere has very few dependencies, which means you can use it pretty much Anywhere, including your legacy application without descending into dependency hell.
